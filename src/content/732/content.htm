732

732 generates a fractal like image. It is just a simple sript that draws multiple circles in a line and rotates them. I dont know why, but 732 gives me the best output. 


    float steps = 2402;
  
    void setup() {
      size(600, 600);
      background(0);
      stroke(255);
      fill(0);
      frameRate(1);
      smooth();
    } 
    
    void draw() {
      float osc = 100;
      translate(width/2, height/2);
      for  (int i = 1; i < steps; i++) {
          rotate(732);
          circle( (width/steps) * i, 0, (width/steps) * i + osc);
        osc += random(-10, 10);
      }
      saveFrame(); 
    }
